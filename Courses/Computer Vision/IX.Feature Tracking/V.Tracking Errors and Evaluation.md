### 1.Error Type

Confusion Matrix:

* TP
* FP
* FN
* TN

### 2.Performance

**True Positive Rate (TPR)/ Recall**

$$\frac{TP}{TP+FN}$$

**False Positive Rate (FPR)**

$$\frac{FP}{FP+TN}$$
**Positive Predictive Value (PPV)/ Precision**

$$\frac{TP}{TP+FP}$$

**Accuracy**

$$\frac{TP+TN}{P+N}$$

### 3.ROC Analysis

>An ROC curve is a graphical plot that illustrates the diagnostic ability of a binary classifier system as its discrimination threshold is varied. It plots the **TPR against the FPR**
