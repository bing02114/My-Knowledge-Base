根据提供的课件 `cv10_-_Feature_Tracking.pdf` 以及近五年（2020-2025）的考试真题，**CV10 (Feature Tracking)** 是考试中关于**动态场景分析**和**系统设计**的核心章节。

这部分内容在考试中通常出现在 **第 3 题 (Motion and Optical Flow / Image Sequence Processing)**，考查形式既有底层的数学推导，也有高层的系统设计。

以下是详细的考点分析及考察形式：

### 1. 核心考点：光流法与 Lucas-Kanade 算法 (Optical Flow & LK Tracker)

这是本章**最核心的数学考点**，经常涉及公式的推导和物理意义的解释。

- **考点内容：**
    
    - **亮度恒定假设 (Brightness Constancy):** $I(x+u, y+v, t+1) = I(x, y, t)$，导出了光流基本方程 $I_x u + I_y v + I_t = 0$ 1。
        
    - **孔径问题 (Aperture Problem):** 单个像素无法确定唯一的运动矢量（1个方程，2个未知数） 2。
        
    - **Lucas-Kanade (LK) 算法：** 引入**空间一致性 (Spatial Coherence)** 假设（邻域内运动相同），通过**最小二乘法 (Least Squares)** 求解位移 $d$。
        
        - 公式：$(A^T A)d = A^T B$ 3。
            
    - **可解性条件：** 矩阵 $A^T A$ 的特征值 $\lambda_1, \lambda_2$ 必须足够大（类似于 Harris 角点检测，意味着该区域有纹理变化，不仅仅是边缘） 4。
        
- **考察形式：**
    
    - **公式填空/解释 (高频):** 题目直接给出最小二乘公式 $(A^T A)d = A^T B$，问最少需要多少点？或者问如何利用 $A^T A$ 的特征值来过滤噪声结果。
        
        - _例如：22-23 Q3(a) 明确考查了光流估计的最小二乘问题，询问最少点数及特征值约束。_
            
    - **孔径问题:** 解释什么是孔径问题，以及如何解决（答案通常是：利用邻域信息/LK算法，或者利用已知摄像机运动）。
        
        - _例如：22-23 Q3(b) 考查了利用已知摄像机平移来解决孔径问题。_
            

### 2. 核心考点：卡尔曼滤波 (Kalman Filter)

这部分内容通常作为**解决遮挡 (Occlusion)** 和 **预测运动 (Prediction)** 的标准答案出现在设计题中。

- **考点内容：**
    
    - **目的：** 将“系统动态模型（预测）”与“测量结果（观察）”结合，以获得更优的估计 5。
        
    - **两个步骤：**
        
        1. **预测 (Prediction):** 基于上一时刻的状态预测当前状态。
            
        2. **校正 (Correction/Update):** 利用当前的观测值更新预测值 6。
            
    - **扩展卡尔曼滤波 (EKF):** 用于非线性系统，通过雅可比矩阵进行线性化 7。
        
- **考察形式：**
    
    - **处理遮挡 (Occlusion Handling):** 在“赛车跟踪”或“气球跟踪”的题目中，几乎必问：“如果物体被遮挡了几帧，如何处理？”
        
        - _回答策略：_ 使用 **卡尔曼滤波器 (Kalman Filter)**。当物体可见时，进行“预测+校正”；当物体被遮挡（无观测值）时，仅依靠“预测”步骤来估计物体的位置，直到它再次出现。
            
        - _例如：21-22 Q2(c)(iii) 和 23-24 Q3(c)(iii) 都问到了如何处理物体消失/遮挡的问题。_
            

### 3. 核心考点：图像金字塔与大位移 (Image Pyramids)

- **考点内容：** LK 算法假设位移很小（Small displacement）。对于大位移，需要构建**图像金字塔 (Image Pyramid)**，从粗糙层（低分辨率）开始跟踪，并将结果传递到精细层 8888。
    
- **考察形式：**
    
    - **理论题:** 解释为什么靠近摄像机的点看起来比远处的点移动得快（运动视差 Motion Parallax），或者如何改进 LK 算法以处理快速运动（答案：多分辨率/金字塔）。
        
        - _例如：22-23 Q3(c)(ii) 问到了为什么近处点移动更快。_
            

### 4. 应用设计：完整跟踪系统 (System Design)

这是将 CV078 (特征检测)、CV09 (特征匹配) 和 CV10 (跟踪) 结合的综合题。

- **考察形式：**
    
    - **流程设计:** “请设计一个系统来跟踪视频中的赛车/气球”。
        
    - **标准回答模板：**
        
        1. **特征提取:** 使用 Harris 或 SIFT 检测显著点。
            
        2. **特征匹配/光流:** 使用 LK 光流法或描述子匹配（如 NNDR）找到帧间对应点。
            
        3. **区分物体:** 使用颜色直方图 (HSV) 或纹理特征区分不同的车/气球。
            
        4. **状态估计与平滑:** 使用 **Kalman Filter** 跟踪位置，利用速度模型预测下一帧位置，并处理短暂的遮挡。
            

### 总结建议 (Summary for Preparation)

针对 CV10 的复习，重点在于理解**数学原理**与**实际应用**的结合：

1. **死磕公式:** 必须理解并记住光流的最小二乘解法 $(A^T A)d = A^T B$，以及矩阵 $A^T A$ 必须可逆且特征值较大的条件。
    
2. **掌握“万能钥匙”:** 遇到“**遮挡 (Occlusion)**”、“**消失 (Disappearing)**”、“**噪声 (Noise)**”这类关键词，直接联想 **Kalman Filter**。
    
3. **理解物理现象:** 理解运动视差（近快远慢），这有助于回答关于光流场解释的理论题。
    
4. **系统思维:** 能够将“Harris角点检测” -> “LK光流跟踪” -> “卡尔曼滤波预测”串联起来，形成一个完整的视觉跟踪方案。
