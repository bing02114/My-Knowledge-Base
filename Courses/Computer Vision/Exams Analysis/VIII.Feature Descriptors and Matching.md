根据提供的课件 `cv09_-_Feature_Descriptors_and_Matching.pdf` 以及近五年（2020-2025）的考试真题，**CV09 (Feature Descriptors and Matching)** 是计算机视觉考试中“应用题”和“流程设计题”的核心来源。

这部分内容通常与特征检测（CV078）一起考查，构成完整的“检测-描述-匹配”流程，主要出现在 **第 2 题（Image Sequence）** 或 **第 3 题（Shape/Recognition）** 中。

以下是详细的考点分析及考察形式：

### 1. 核心考点：SIFT 描述子 (SIFT Descriptor)

SIFT 是最经典的特征描述子，考试中常考查其构建步骤及不变性原理。

- **考点内容：**
    
    - **构建步骤：**
        
        1. **检测：** 使用 DoG 极值检测关键点（CV078 内容）。
            
        2. **主方向分配 (Orientation Assignment):** 计算邻域梯度直方图，找到峰值作为主方向（实现旋转不变性） 1。
            
        3. **描述子生成 (Descriptor Generation):** 在 $16 \times 16$ 的邻域内，将其划分为 $4 \times 4$ 的子区域（Cells），每个子区域计算 8 个方向的梯度直方图。最终得到 $4 \times 4 \times 8 = 128$ 维向量 2。
            
    - **不变性：** SIFT 对旋转、缩放、光照变化具有鲁棒性。
        
- **考察形式：**
    
    - **设计题:** 在“寻找 Big Ben”或“跟踪气球”的题目中，问“如何匹配不同视角下的物体？”。
        
        - _回答策略：_ 使用 SIFT 描述子，因为它们对旋转和缩放具有不变性（Rotation and Scale Invariant）。需简述 SIFT 描述子的构建过程（128维向量）。
            
    - **细节追问:** 为什么要归一化描述子向量？（为了抵抗光照变化）。
        

### 2. 核心考点：SURF 描述子 (SURF: Speeded Up Robust Features)

作为 SIFT 的快速版本，常作为替代方案或对比项考查。

- **考点内容：**
    
    - **原理：** 使用 Haar 小波响应计算特征 3。
        
    - **积分图 (Integral Image):** SURF 的核心加速机制，利用积分图可以在常数时间内计算任意矩形区域的像素和 4。
        
    - **描述子维度：** 通常为 64 维（$4 \times 4$ 子区域，每个区域 4 个值），比 SIFT 更紧凑。
        
- **考察形式：**
    
    - **概念解释:** 解释什么是积分图，以及它如何加速特征计算。
        
    - **选择题/对比题:** 为什么 SURF 比 SIFT 快？（因为使用了积分图和 Haar 小波近似）。
        

### 3. 核心考点：特征匹配策略 (Feature Matching)

描述子提取出来后，如何进行匹配是关键步骤。

- **考点内容：**
    
    - **最近邻距离比 (Nearest Neighbour Distance Ratio, NNDR):** 这是最常用的匹配策略。计算最近邻距离 $d_1$ 和次近邻距离 $d_2$ 的比值 $NNDR = d_1 / d_2$。如果 $NNDR < 0.8$，则认为是正确匹配；如果接近 1，则说明匹配模棱两可，应拒绝 5。
        
    - **距离度量：** 欧氏距离（Quadratic distance）或余弦相似度（Cosine similarity） 6。
        
- **考察形式：**
    
    - **应用题:** 在匹配题目中，问“如何去除错误的匹配点？”。
        
        - _回答策略：_ 使用 NNDR 策略。解释如果一个特征点在另一幅图中有两个非常相似的匹配点（歧义），NNDR 会接近 1，从而将其过滤掉，保留独特性高的匹配。
            
        - _例如：23-24 Q3(b) 询问如何匹配 Big Ben，你需要提到使用 SIFT 特征加 NNDR 匹配策略。_
            

### 4. 辅助考点：形状上下文 (Shape Context)

- **考点内容：** 使用对数极坐标直方图（Log-polar histogram）来描述形状 7。
    
- **考察形式：** 较少单独考查，但在形状识别题目中可以作为一种替代 SIFT 的形状描述符提及。
    

### 总结建议 (Summary for Preparation)

针对 CV09 的复习，建议关注“描述”与“匹配”两个环节：

1. **描述子 (Description):**
    
    - **SIFT:** 牢记 $4 \times 4 \times 8 = 128$ 维向量的由来，以及主方向分配的作用（旋转不变性）。
        
    - **SURF:** 记住关键词“积分图”和“Haar 小波”，这是它速度快的核心。
        
2. **匹配 (Matching):**
    
    - **NNDR:** 务必掌握 $d_1/d_2 < Threshold$ 这个策略，它是过滤误匹配的黄金法则。
        
3. **场景应用:**
    
    - 遇到“视角变化”、“旋转”、“缩放”等关键词 $\rightarrow$ 必选 SIFT/SURF。
        
    - 遇到“实时性要求高” $\rightarrow$ 优先选 SURF 或 BRIEF（虽然课件未深究，但 SURF 是课件重点）。
